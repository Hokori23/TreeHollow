# 树洞

初始化

- 安装 npm 依赖

```bash
npm i @commitlint/cli @commitlint/config-conventional -g
npm i
```

- 安装 vscode 推荐扩展

git 提交

> 提交规范请参考`./.commitlintrc.js`和`.scripts/pre_commit.js`

```bash
正确示例：
  eg1: git commit -m 'fix: 修正xxx'
  eg2: git commit -m 'feat: 1 实现xxx功能 2 ....'
```
